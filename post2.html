Portfolio I: Reflections

Reflecting on how I designed my website...

Posted by Anna Heckadon on September 26, 2017

I used a template for designing the basics of my website and then inputed my own words/photos. In particular I changed the layout of the "Contact" page. I changed the "Contact" page to "Social Media" because I'm not personally too keen on putting my email up for everyone to see. Maybe I'll change my mind later though and put it up. We'll see. I also added a comment function with the help of the internet! I like the idea of my followers being able to communicate with me if they wish to!

Nothing really went wrong. Nothing big at least. I had some issues with the images, because they automatically resize themselves to fit in the space set aside for them. Unfortunately, that's just how the template is set up, so I dealt with it. For some pages, I had to choose a different image than I wanted so that it would fit well and look good. Additionally, the images get cropped differently whether you're on a smartphone or a computer. I chose to set them up to look best when viewing on a computer. However, if I learn more CSS and figure out how to edit the template effectively, then maybe I could change how the photos are sized and fix up the difference between computer and smartphone viewing. That could take awhile, so for now things are fine as is.

My website is successful because I organized the layout so that it is easy to follow and so that it looks professional. Also, I feel as though I have made massive progress at grasping the coding language HTML through making this website. I know a lot now about how HTML works through studying the code for the original text from the template.

I have had so much fun making this website. :) If you'd like, please comment with any suggestions on how you think I can make it even better!! All ideas are welcome!
